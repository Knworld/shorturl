URL SHORTENER

  • Pass a URL as a parameter and receive a shortened URL
  • The shortened URL will redirect to the original link

Example Input:

https://kn-short-url.herokuapp.com/short/http://www.google.com

Example Output:

{"original":"http://www.google.com", "shortened":"https://kn-short-url.herokuapp.com/H8ncxQGXW4"}

Usage

https://little-url.herokuapp.com/H8ncxQGXW4 --- redirects to --> http://www.google.com
