URL SHORTENER

  • Pass a URL as a parameter and receive a shortened URL
  • The shortened URL will redirect to the original link

Example Input:

https://little-url.herokuapp.com/new/https://www.google.com

Example Output:

{"original":"http://www.google.com", "shortened":"https://little-url.herokuapp.com/xxxx"}

Usage

https://little-url.herokuapp.com/xxxx --- redirects to --> http://www.google.com
